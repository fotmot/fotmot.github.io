Фоторамка для Яндекс.Диска с поддержкой Motion Photo

Подсказки:
  • Дважды тапните(кликните) по экрану, что бы перейти в полноэкранный режим
  • Для удаления фотки с диска нажмите первую конпку (Мусорный бак)
  • Для остановки смены фоток нажмите вторую кнопку (Пауза)
  • Для переключения на следующую фотку нажмите третью кнопку (Перемотка вперед)
  • Для смены папки, откуда показывать фотки, нажмите четвертую кнопку (Папка). Внимание, фотки загружаются только из корня папки. Поиск файлов в подпапках не осуществляется.
  • Подогнать фото под экран можно с помощью пятой кнопки (Распирающие стрелки)
  • Отключить Motion Photo и автозапуск видео можно шестой кнопкой - (Картина)
  • Дать автору на лимонад - можно тут
  • Обратная связь
  • Выход из аккаунта
